## 制作备注

**音乐建议**：交给 Suno 生成，关键词建议用——`noir jazz, suspenseful piano, cinematic thriller, slow build to intense climax, 90 seconds`

**素材优先级**：
1. 镜 05、10、15 段是视觉核心，优先用可灵/Hailuo 生成动态镜头
2. 角色卡片（镜 06-09、11）用 MidJourney 出静帧即可，加入轻微放大动效
3. 游戏 UI 演示（镜 12-13）直接录屏或做模拟截图

**最关键的一句旁白**：镜 18-19 是整个 PV 的情绪反转点，建议用 ElevenLabs 真人配音，不要用 TTS 机器音。
