## 3.	产品形态
## 3.1 题材、美术、世界观
### 3.1.1 题材
《逆转侦探》是一款以“扮演凶手而不是侦探”为核心反转卖点的悬疑犯罪推理游戏，题材上融合了犯罪模拟、沉浸式叙事、心理博弈与黑色电影风格的现代都市惊悚。玩家并非以传统侦探视角搜证破案，而是站在凶手的立场提前规划、布置、误导和脱身，在一步步执行犯罪方案的过程中反向理解侦探如何拆解案件。

从体验形态上看，本作并不是纯文字 AVG，也不是纯沙盒潜行，而是“AI 叙事驱动 + 2.5D 场景交互”的混合型产品：案件背景、人物关系、场景推进和结局分支由大模型生成，玩家则通过图形化界面在空间中移动、观察、接近物件、执行动作，再在关键节点把局面交给 AI 裁定。题材吸引力不在血腥刺激，而在于“如何完成一场看似无懈可击的犯罪”以及“玩家在不断代入凶手思维的过程中，仍能否保住侦探的理性与正义”。

整体气质应保持克制、冷峻、危险感和反转感，强调高智商犯罪、现场伪装、证据处理、时间差利用和人物心理博弈，而不是动作冒险或正面对抗。玩家获得的核心快感，是用凶手的思维完成布局，再用侦探的理解反过来验证这套布局是否真的成立。

### 3.1.2 美术
### 3.1.3 世界观
《逆转侦探》是一款沉浸式虚拟现实游戏，玩家需要在游戏中扮演凶手，亲历犯罪全过程，完成作案后，游戏系统会从计划周密性、现场清洁度、误导有效性、心理状态等多个维度评分，判定玩家是否实现完美犯罪。[玩家]从小就梦想着成为一名大侦探，为了锻炼自己的能力，他频繁游玩《逆转侦探》，力图通过实现完美犯罪来理解凶手的心理和行为，在现实中更好地破案。
随着[玩家]扮演凶手次数的增多，他越来越了解凶手犯案的动机和手法，也终于成为专业侦探，经手许多复杂的案件。为了拆解凶手的布局，他在《逆转侦探》中重新演绎现实中的案件，试图找到突破的契机。
《逆转侦探》的游戏制作人江川是多年前享誉全球的传奇侦探，破案率接近百分百，但他在四十七岁时突然退隐，下落不明。事情的真相是：江川拥有病理性的强共情能力，每侦破一个案件，他都离凶手更近一些，二十年的积累让他的精神世界变成了一座住满凶手的监狱，再接触罪案，他就无法控制自己的犯罪欲望了。因此他制作了名为《逆转侦探》的游戏，想要筛选出同样拥有强共情能力、能够代入凶手思维却不被吞噬的人，在自己成为凶手、危害社会时能够挺身而出制伏自己。他提取出自己接近凶手的那部分人格，创造了游戏中的侦探AI“零”，它拥有完整的犯罪人格，因此能生成极其真实的罪案模拟。当“零”筛选出符合条件的人时，它会将江川的真实目的告知此人，逼迫他成为江川的对手——没有别的选择。
[玩家]就是这个特殊的人，在长期扮演凶手、体验犯罪心理后，仍能保有清醒正义的自我。在[玩家]了解江川的故事后，江川也终于无法抑制犯罪人格，在现实中成为超强罪犯，在游戏和现实中，[玩家]都将与江川展开对抗……


## 3.2 核心玩法机制
《逆转侦探》的核心玩法以“确定性准备 + 关键决策推进”为骨架，形成一条从探索、布局到 AI 裁定的闭环。玩家每一轮的目标不是立刻行凶，而是先在当前场景中完成尽可能多的可确定准备动作，再在真正改变风险、时间、地点和证据结构的关键节点做出抉择。

具体流程如下：

1. **开局生成案件与初始场景**  
系统基于预设故事、玩家身份、受害者关系与动机，生成本局的初始场景 JSON，包括背景、NPC、可互动物件、可执行选项和当前叙事状态。玩家进入的不是固定脚本关卡，而是一组由 AI 与规则共同搭建的犯罪模拟空间。

2. **2.5D 场景探索与接近交互**  
玩家在 2.5D 场景中控制角色移动。靠近可互动物件后，界面弹出该物件当前可执行的动作选项，例如观察、取用工具、确认动线、布置伪装、制造不在场证明等。NPC 在 Demo 中默认以静态站位为主，必要时可沿闭环路径进行简单巡逻。

3. **本地规则处理“确定性准备动作”**  
大多数准备动作属于 `local_rule`，由客户端依据状态条件即时处理，不必立刻请求 AI。这类动作的价值在于稳步改变场景状态，例如解锁门、拿取工具、确认监控盲区、提前安排掩护、记录证人动线等。玩家通过这些确定性动作为后续关键决策创造条件，并积累局面优势。

4. **关键决策触发 AI 结算推进**  
当玩家选择被标记为 `immediate_ai` 的关键动作时，例如直接行凶、提前实施危险方案、把怀疑转嫁给他人，或提交不在当前选项内的“自由行动”，系统才会把【初始设定 + 当前完整历史记录 + 本轮动作】发送给 AI。AI 返回新的场景、叙事、状态和后续局面，决定故事是否继续推进、是否切换场景，或是否直接进入结局。

5. **自由行动补足玩家创造力**  
除了系统给出的显式选项，玩家还可以主动输入自由行动，描述一个不在当前可见选项中的计划。自由行动天然属于高风险、高表达度的玩法入口，适合在常规动作无法覆盖玩家意图时使用。它既强化了“犯罪模拟”的自由度，也为 AI 的动态裁定提供了更强的变体空间。

6. **多分支结局与结果反馈**  
AI 每次返回都会给出 `game_status`，用于判定玩家当前是继续推进、达成胜利、彻底失败，还是进入特殊结局。结局并非单纯的数值结算，而是结合玩家此前布置、清理、误导和临场决策形成的叙事结果。理想体验是：玩家既能从成功脱身中获得“完美犯罪”的快感，也能从失败复盘中反向理解侦探视角如何破解自己。

因此，本作的核心玩法不是单次点击某个“杀人”选项，而是围绕以下循环展开：

**场景探索 → 确定性准备 → 制造条件差 → 触发关键决策 → AI 裁定推进 → 进入下一轮布局或结局**

这个循环使游戏同时具备三层吸引力：  
其一是空间中的交互和试探；  
其二是规则层面的布局与博弈；  
其三是 AI 带来的剧情反应、分支推进与结局变化。

## 3.3 玩家长期参与方式
### 3.3.1主线与DLC长期更新
以章节形式持续释出《逆转侦探》的主线剧情，定期更新付费/免费DLC，保持玩家对核心叙事的长期关注。章节结尾设置悬念钩子，维持玩家追更动力；关键剧情节点提供分支选择，影响非关键剧情走向；章节完成后解锁专属称号与剧情画廊。
DLC拓展方向：（1）角色DLC：《江川篇：归零之前》，江川视角前传。（2）案件包DLC：独立于主线的案件，比如赛博朋克背景、古代背景案件集、节日特别篇等。（3）PVP模式DLC：一名玩家扮演凶手，一名玩家扮演侦探，裁判AI负责评分。
### 3.3.2 内置UGC模块，开放玩家创意工坊
玩家可以上传原创案件至创意工坊，游戏内置大模型将依据玩家上传的案件生成游戏场景、互动和文字剧情，玩家试玩调整并通过社区审核后公开上架。
